# Cartify

*Cartify* is a responsive E-Commerce website built using *HTML, CSS, and JavaScript*. The platform allows users to browse a variety of products with an intuitive interface and search functionality, providing a seamless shopping experience.

---

## Features

- Responsive design for desktop and mobile devices  
- Dynamic product display  
- Product search functionality  
- Clean and modern user interface  
- Multiple product categories  

---

## Technologies Used

- *HTML* – Structure of the website  
- *CSS* – Styling and responsive design  
- *JavaScript* – Interactive features like product search
